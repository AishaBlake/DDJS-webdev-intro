# Web Devlopment Workshop
##HTML HTML stands for HyperText Markup Language. It is the "hidden" code that we use to communicate with others on the World Wide Web. When we write HTML code, we add "tags" that help to create the structure of the page; tags tell the web browser just how to display the content of the html document.
### Syntax Syntax is another word for grammar. Because computers are very literal machines, we must use "good" syntax in our code.
### Specific Tags Tags are angled brackets that look like this <... >, with some letters or word inside them. These brackets are on your keyboard, over the comma and the period. We type words into these brackets to tell the computer what to do.
- The `div` tag defines a division or section in an HTML document - The `p` tag defines a paragraph - The `img` tag defines an image - The `a` tag defines a hyperlink - The `ul` tag defines an unordered list - The `li` tag defines a list item

##CSS
### The CSS Rule Set
#### Selector We use the selector to tell the computer what part (or parts) of our HTML we want to change with this CSS rule. A selector can be as simple as using the name of a tag or it can be very complex and specific. ```css div { } ```
#### Property The property is the thing about the selected element (or elements) that we want to change. There are _tons_ of these and I don't expect you to remember most of them. We'll talk about some very commonly-used ones to get you started. ```css div { background-color: } ```
#### Value If the property is the aspect we want to change, the value is what we want to change it _to_. ```css div { background-color: blue; } ```
#### Bringing It All Together - There's a colon (`:`) between properties and their values. - Property/value pairs are separated with a semi-colon (`;`). - We can have as many property/value pairs as we want, but we have to surround them with curly braces (`{}`). ```css div { background-color: blue; color: limegreen; font-size: 20px; } ```
### background
#### What it Does The `background` property is a quick way to change lots of different things related to the background of our web page's elements. ```css div { background-color: blue; } ```
### color
#### What it Does The `color` property changes the color of the _text_ inside the selected element. ```css div { color: limegreen; } ```
### width
#### What it Does We can use the `width` property to change the width of elements. By default, block elements like `p`s and `div`s will take up the full width of their parent elements. ```css div { width: 500px; } ```
### font-size
#### What it Does The `font-size` property affects the size of text. ```css div { font-size: 20px; } ```
