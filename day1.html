### The CSS Rule Set
#### Selector We use the selector to tell the computer what part (or parts) of our HTML we want to change with this CSS rule. A selector can be as simple as using the name of a tag or it can be very complex and specific. ```css div { } ```
#### Property The property is the thing about the selected element (or elements) that we want to change. There are _tons_ of these and I don't expect you to remember most of them. We'll talk about some very commonly-used ones to get you started. ```css div { background-color: } ```
#### Value If the property is the aspect we want to change, the value is what we want to change it _to_. ```css div { background-color: blue; } ```
#### Bringing It All Together - There's a colon (`:`) between properties and their values. - Property/value pairs are separated with a semi-colon (`;`). - We can have as many property/value pairs as we want, but we have to surround them with curly braces (`{}`). ```css div { background-color: blue; color: limegreen; font-size: 20px; } ```
